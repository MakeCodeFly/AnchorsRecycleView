# AnchorsRecycleView
锚点，高仿支付宝锚点功能，由支付宝首页->更多->进入我的应用的页面动画效果和逻辑


仿支付宝更多应用，效果仿支付宝更多进入的城市服务栏目的刷新切换Tab效果，仿H5锚点开发，上下滑动横向移动tab，及往上滑动Tab栏目悬浮顶部以及可以拖拽、移动等编辑头部itemview，用的是RecycleView+TabLayout实现的
